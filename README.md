# Personal Portfolio Website

A responsive personal portfolio website showcasing my skills, projects, technical experience, and contact information.

## Live Website

[View Portfolio](https://jawahar08.github.io/jawahar-portfolio/)

## Features

* Responsive design for desktop and mobile devices
* About Me section
* Technical Experience section
* Projects showcase
* Contact section
* GitHub and LinkedIn integration
* Downloadable Resume/CV

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## Projects Featured

### Memory Firewall

AI security framework for protecting long-term memory in AI agents using claim extraction, contradiction detection, provenance tracking, risk scoring, and quarantine workflows.

IoT-based attendance management system using Arduino Uno, RFID technology, and Google Sheets integration for real-time attendance tracking.
